# 01_python_numpy_essentials

## Назначение папки

Демонстрация базовых навыков работы с NumPy для обработки 3D медицинских данных (КТ почек).  
Скрипты показывают разницу в производительности между циклами Python и векторизованными операциями NumPy, а также механизм broadcasting.

## Содержание

| Файл | Описание |
|:---|:---|
| `vectorization_demo.py` | Сравнение SSD (Sum of Squared Differences) через тройной цикл Python и векторизацию NumPy. Оценка времени для реального КТ 512³. |
| `broadcasting_demo.py` | Демонстрация broadcasting: применение маски к 3D объему, нормализация срезов, центрирование данных. Сравнение с циклами. |
| `performance_comparison.py` | Сводный бенчмарк всех операций (SSD, маска, нормализация) в одной таблице. Оценка клинической применимости. |

## Ключевые выводы

- **Векторизация NumPy** быстрее циклов Python в **100–1000 раз** для операций над 3D массивами
- **Broadcasting** ускоряет работу с массивами разной формы в **10–50 раз** без копирования памяти
- Для реального КТ почки (512×512×300 = 78 млн вокселов) все операции укладываются в **<10 секунд**
- Это необходимое условие для достижения **УГТ-3** (клинически пригодный прототип)
