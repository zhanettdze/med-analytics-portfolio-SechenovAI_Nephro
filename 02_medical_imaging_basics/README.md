# 02_medical_imaging_basics

## Назначение

Демонстрация базовых навыков работы с медицинскими форматами данных (DICOM, NIfTI), понимание единиц Хаунсфилда (HU) и фаз контраста КТ почек.

## Содержание

| Файл | Описание |
|:---|:---|
| `01_dicom_to_numpy.py` | Загрузка DICOM серии в 3D numpy массив |
| `02_nifti_io.py` | Чтение и запись NIfTI файлов |
| `03_hounsfield_units.py` | Конвертация в HU, оконтуривание, диапазоны для почек |
| `04_kidney_segmentation_demo.py` | Сегментация почки по HU + постобработка |
| `05_contrast_phases.py` | Фазы контраста (артериальная, паренхиматозная, экскреторная) |
| `06_visualize_ct.py` | Визуализация срезов КТ и масок |
