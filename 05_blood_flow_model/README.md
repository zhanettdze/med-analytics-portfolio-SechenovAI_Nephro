## Методы оценки кровотока

1. **Kety model**: перфузионная модель на основе динамики контраста
2. **Jacobian-based**: оценка через поле деформации (det(J) > 1 → увеличенный кровоток)
3. **Deconvolution**: метод деконволюции с артериальной входной функцией (AIF)
