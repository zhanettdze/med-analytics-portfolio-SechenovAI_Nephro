# med-analytics-portfolio

## Описание репозитория

Данный репозиторий представляет собой портфолио для стажировки «Аналитик медицинских данных» в проекте SechenovAI_Nephro.

Репозиторий содержит учебные и демонстрационные проекты, выполненные в процессе подготовки к стажировке. Содержимое репозитория отражает текущий этап освоения инструментов и методов, необходимых для решения задач медицинской аналитики.

## Структура репозитория

| Папка | Содержание |
|-------|------------|
| `01_python_numpy_essentials/` | Примеры работы с NumPy: векторизация, матричные операции |
| `02_medical_imaging_basics/` | Базовые операции с медицинскими изображениями (DICOM, HU) |
| `03_validation_metrics_basics/` | Реализация метрик для оценки качества моделей |
| `04_image_registration/` | Методы регистрации медицинских изображений |
| `05_blood_flow_model/` | Моделирование почечного кровотока |
| `06_3d_visualization/` | Трехмерная визуализация органов и результатов регистрации |
| `data/` | Инструкции по получению тестовых данных |
| `*/extra/` | Дополнительные учебные проекты |

## Содержание по папкам

### 01_python_numpy_essentials

- векторизация вычислений в сравнении с циклами
- матричные операции, broadcasting
- утилиты для работы с массивами медицинских данных

### 02_medical_imaging_basics

- загрузка DICOM и NIfTI через SimpleITK
- вычисление статистики HU внутри области интереса
- визуализация срезов

### 03_validation_metrics_basics

- Dice coefficient для сегментационных масок
- MAE, RMSE, MAPE для регрессионных задач
- коэффициент корреляции Пирсона
- график Бланда-Альтмана для сравнения двух методов измерения

### 04_image_registration (в разработке)

- жесткая (rigid) регистрация
- аффинная (affine) регистрация
- нелинейная регистрация (B-spline, демоны)
- оптимизация метрики взаимной информации

### 05_blood_flow_model (в разработке)

- модель перфузии Кити
- расчет кровотока на основе динамики контраста
- связь с полем деформации при регистрации

### 06_3d_visualization (в разработке)

- построение поверхности из сегментации
- визуализация полей деформации
- наложение зарегистрированных объемов

### data

Папка содержит данные.
