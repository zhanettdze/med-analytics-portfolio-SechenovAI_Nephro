# Data

## Открытые датасеты для тестирования

### KiTS19 (Kidney Tumor Segmentation)
- URL: https://kits19.grand-challenge.org/
- Скачать 2-3 пациента (например, case_00000, case_00001)
- Разместить в папке `data/case_00000/imaging.nii.gz`

## Примечание
- Файлы с данными не хранятся в репозитории
- Все данные используются только в учебных целях
